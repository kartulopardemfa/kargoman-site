Support

If you need help with KargoMan, the fastest support route is email.

How to get help

Please include the device model, iOS version, game version, and a short description of the issue. If the problem is visual, include a screenshot.

Email Support

Email

If you need a direct support contact for store review or user support, email umayedin@icloud.com.

Useful details to include

  • What you expected to happen
  • What actually happened
  • Whether the issue happens every time or only sometimes
  • Any screenshots or steps that reproduce the bug

Report an ad issue

If you see an inappropriate, age-inappropriate, or broken ad in KargoMan, report it by email and include the ad type, the screen where it appeared, the approximate time, and a screenshot if possible.

Report Ad Issue

Common topics

  • Ads not loading or reward not applying
  • Inappropriate or age-inappropriate ads
  • Language issues in the international build
  • Visual bugs in the label, invoice, or result screens
  • Progress or balance behaving unexpectedly
